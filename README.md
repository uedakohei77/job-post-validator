# Job Post Validator

A tool to fetch and validate job postings, ensuring they meet specified criteria and formatting standards.

## Project Structure

- `config/`: Authentication and configuration files.
- `src/`: Fetching and validation scripts.
- `requirements.txt`: Python dependencies.

## Setup

1. Create a virtual environment:
   ```bash
   python -m venv .venv
   source .venv/bin/activate
   ```
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Set up your environment variables in a `.env` file (see `.gitignore` for excluded files).
